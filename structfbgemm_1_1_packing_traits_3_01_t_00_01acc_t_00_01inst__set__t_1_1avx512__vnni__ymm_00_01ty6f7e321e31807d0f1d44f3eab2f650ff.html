Detailed Description

template<typename T, typename accT>
struct fbgemm::PackingTraits< T, accT, inst_set_t::avx512_vnni_ymm, typename std::enable_if< is_8bit< T >::value &&is_16or32bit< accT >::value >::type >

Since there is no int16_t accumulation for AVX512 VNNI, we redirect int16_t to int32_t accumulation and use the same blocking parameters as int32_t.

This is picked when T is of int8 type (signed or unsigned) and instruction set is avx512_vnni_ymm.
